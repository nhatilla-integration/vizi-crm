# Vizi CRM

Vizi CRM é uma aplicação desenvolvida para atuar como uma camada analítica e organizacional dentro do CRM, auxiliando empresas na tomada de decisões estratégicas e operacionais.

A plataforma funciona como uma “peneira inteligente”, analisando minuciosamente os dados inseridos no sistema para identificar padrões, gargalos e oportunidades de melhoria. O objetivo é ajudar empresas a entenderem quais ações estão gerando bons resultados e quais processos precisam ser ajustados quando os números não estão performando como esperado.

O projeto está sendo desenvolvido utilizando React.js e Tailwind CSS, com foco em organização visual, experiência do usuário e escalabilidade da aplicação.

---

## Tecnologias utilizadas

* React.js
* JavaScript
* Tailwind CSS
* HTML5
* CSS3

---

## Como executar o projeto

```bash
npm install
npm start
```

O projeto será iniciado em:

```bash
http://localhost:3000
```
